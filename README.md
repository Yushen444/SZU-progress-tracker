# SZU Progress Tracker

一个面向深圳大学学生的培养方案导入、课程管理与毕业进度查询工具。它将不同年级、学院和修读类型的培养方案整理为可直接导入的课程数据库，帮助学生更直观地记录课程、成绩、绩点和毕业学分进度。

![SZU Progress Tracker 首页预览](assets/szu-progress-tracker-preview.png)

## 在线体验

**[打开网页体验](http://yumao.japaneast.cloudapp.azure.com/)**

建议首次使用时先导入主修培养方案；之后可在“选课与成绩”和“通关总览”中持续维护个人进度。

## 主要功能

- 导入本地 DOC、DOCX、RTF、PDF 格式的培养方案。
- 从云端培养方案库按年级、学院、修读类型和关键词筛选并导入。
- 支持主修与辅修方案：主修决定毕业总学分与模块要求，辅修独立计算进度。
- 课程按学期管理，支持拖动排序、自定义课程和从课程库选课。
- 支持等级制、百分制、含附加题和不计绩点四种成绩录入方式。
- 自动计算已获得学分、在修学分、平均学分绩点及各分类要求完成情况。
- 可单独管理、移除误导入的辅修方案，不影响主修方案。

## 培养方案数据

当前公开数据库包含：

| 年级 | 培养方案数量 |
| --- | ---: |
| 2025 级 | 328 |
| 2024 级 | 354 |
| 2023 级 | 357 |
| 合计 | 1,039 |

共包含 110,187 条课程记录。数据来源于深圳大学官网已发布培养方案的详情数据；同名方案可能归属不同学院，因此以官网方案标识作为唯一来源标识。

## 本地运行

需要安装 Node.js。克隆仓库后，在项目目录执行：

```powershell
npm run serve
```

然后访问 <http://127.0.0.1:4173>。

## 项目结构

```text
site/
  index.html        # 单页网页
  plan-db/          # 可直接使用的培养方案数据库
scripts/
  serve.mjs         # 本地静态服务
  test-*.mjs        # 页面与主辅修进度回归检查
```

## 数据与隐私说明

- 本仓库只包含网页运行所需的公开培养方案数据。
- 原始抓取数据、服务器快照、部署文件、历史备份和任何私钥均不会提交。
- 课程与成绩记录仅保存在使用者自己的浏览器本地存储中，不会由本项目上传或收集。
